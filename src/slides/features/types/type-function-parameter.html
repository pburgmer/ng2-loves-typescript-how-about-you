Type Inferencing bei Funktionsparametern
  • Für alleinstehende Funktionen nicht möglich
    • Parameter ohne expliziten Typ sind any
    • Kann per Konfiguration verboten werden (noImplicitAny)
    // 'a' and 'b' are both of type any: a explicit, b implicit
function foo(a: any, b) {}
  • Type-Inferencing bei Lamdas möglich 
    // type of result defined by signature of bar
bar(function callback(result) {});
